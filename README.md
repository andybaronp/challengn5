# Challenge

 ### Challenge de productos

## Caraterísticas del challenge

- Lista productos
- Agregas productos al carrito
- Eliminar productos del carrito
- Comprar productos
- Vaciar carrito
 

## Se utilizó:

- React
- uuid para ids unicos
- sass
- JSON como data iniciarl
- localstorage
 ## Probar el challenge

 - git clone https://github.com/andybaronp/challengn5
 - cd challengn5
 - npm install
 -npm run dev
